### Datos

| | |
|---|---|
| **Población** | Monistrol de Montserrat |
| **Dirección** | Passeig Germans la Salle, s/n · 08691 |
| **Coche** | ~**17 min** monestir · ~**19 min** Can Prat |
| **Horario** | **11:00–20:00** · temporada ~jun–sep |
| **Precio adulto 2026** | Lab **8 €** · sáb/fest **9 €** |
| **Acceso** | Entrada de día en taquilla · **no hace falta abono** |
| **Parking** | Gratis · reseñas OK |
| **Tel. piscina** | 673 121 337 |
| **Web** | [Ayuntamiento](https://www.monistroldemontserrat.cat/serveis-municipals/179-equipaments/1834-piscina-municipal.html) · [tarifas 2026](https://www.monistroldemontserrat.cat/continguts/opertura-de-la-piscina-municipal-temporada-2026.html) |

### Comer · picnic · sombra

| Pregunta | Respuesta |
|----------|-----------|
| ¿Llevar **nuestra comida** dentro? | **No lo deis por hecho.** Picnic a veces cerrado. **Preguntad en taquilla.** |
| ¿**Picnic con sombra** dentro? | Irregular. Plan B fuera ★ |
| ¿**Bar** dentro? | Según temporada · confirmad |
| ¿Comer **cerca** fuera? | Sí ★ **Parc de l'Alvèrnia** · o **Bresquet** · o pueblo |

**Plan comida Monistrol:**

1. Baño · bar si hay · **o**  
2. Picnic ★ **Parc de l'Alvèrnia** · **o**  
3. **Bresquet** / pueblo  

### Picnic cerca *(fuera)*

| Sitio | Distancia | Qué hay | Sombra |
|-------|-----------|---------|--------|
| **Parc de l'Alvèrnia** ★ | ~**5–10 min** a pie / **~2 min** coche | **2 mesas** picnic · parking · parque infantil | Sí |
| **Drecera dels Tres Quarts** | ~**5–8 min** coche (BP-1121) | 2 mesas · parking | Variable |
| **Santa Cecília** | ~**15–16 min** coche | 4 mesas · vistas | Parcial |

**Maps:** *Parc de l'Alvèrnia Monistrol* / *área autocaravanes Monistrol* (C-55).  
[Detalle ayuntamiento](https://www.monistroldemontserrat.cat/serveis-municipals/179-equipaments/area-autocaravanes.html)

**Cuándo:** tras Turó Ermità / mañana monestir.

## 2. Collbató ★ picnic + entrada sin abono

### Datos

| | |
|---|---|
| **Población** | Collbató *(cara sur · cuevas)* |
| **Coche** | ~**25–30 min** desde monestir |
| **Horario jul–ago tip.** | **11:00–20:00** · jun/sep a veces hasta 19:30 |
| **Precio adulto 2026** | Lab **7 €** · fest **8 €** · jóvenes 15–25: 5 / 7 € ([díptico](https://www.collbato.cat/files/doc7318/diptic-print.pdf)) |
| **Acceso** | ✅ **Entrada puntual** en la piscina · **no hace falta abono de temporada** |
| **Findes** | A veces **tope de entradas** (p. ej. ~150/día) → id temprano |
| **Web** | [collbato.cat](https://www.collbato.cat/) · tel. ayuntamiento **93 777 01 00** |

### Comer · picnic · sombra ★

| Pregunta | Respuesta |
|----------|-----------|
| ¿Llevar **nuestra comida**? | ✅ **Sí** · comer/beber en la **zona picnic** (no en el resto del recinto) |
| ¿**Picnic con sombra**? | ✅ Zona picnic habilitada · bar abierto en horario de piscina |
| ¿**Bar** dentro? | ✅ Todo el horario de apertura |
| ¿Sombrilla / silla propia? | ⚠️ A menudo **no** (salvo +60 / movilidad / embarazadas) · usad sombra del recinto |

**Plan comida Collbató:**

1. Tupper + nevera soft (**sin cristal**).  
2. Comer en **zona picnic**.  
3. Bar para bebida / snack si queréis.

### Instalaciones / opiniones

- Piscina municipal de pueblo · familiar  
- Bar + picnic claros en normas del ayuntamiento  
- Findes puede llenarse · ir pronto  

**Cuándo:** día de calor / tras cuevas Collbató · media mañana.

| Situación | Elegid |
|-----------|--------|
| Ruta acabada ~11:00 · cerca · vistas | **Monistrol** |
