# Ruta · Sant Jeroni *(cim de Montserrat)* ★ obligatoria

Cima más alta de Montserrat · **1236 m**.  
Wikiloc referencia: [Sant Jeroni – cim](https://es.wikiloc.com/rutas-senderismo/sant-jeroni-el-cim-mes-alt-de-la-muntanya-de-montserrat-6700944)

**Plan elegido:** Monestir → **Funicular Sant Joan** → caminar a Sant Jeroni → volver (funicular o a pie).

## Funicular de Sant Joan ★

| | |
|---|---|
| **Qué es** | Funicular del **Monestir** → **Sant Joan** (arriba). **No** es el cremallera. |
| **Duración** | ~**6–7 min** |
| **Horario verano tip.** | **~10:00 – 18:30** · cada ~12–20 min *(confirmar el día)* |
| **Primera subida útil** | Estar en cola **~09:45–10:00** |
| **Última bajada** | No apuréis: dejad margen (cima + vuelta + cola) · meta práctica bajar funicular **antes 17:00–17:30** si podéis |
| **Precio tip.** | Sencillo adulto ~**5,70 €** (e-ticket web) · ida/vuelta ~**doble** o pack · **comprobar** [vol.cremallerademontserrat.cat](https://vol.cremallerademontserrat.cat/) / taquilla |
| **Dónde comprar** | Taquilla estación inferior (junto monestir) **o** e-ticket online |
| **Recomendación billete** | **Ida y vuelta** si volvéis por el mismo sitio · **solo ida** si bajáis andando al monestir por Camí Vell |

⚠️ **No podéis coger el funicular a las 8:00** — abre ~**10:00**. El madrugón sirve para desayunar con calma y llegar a la primera.

## Logística del día ★ *(ajustada a horario real)*

Propuesta vuestra: despertar 7:00 · desayuno 7:30 · salir 8:00.  
**Encaje real con funicular 10:00:**

| Hora | Qué |
|------|-----|
| **07:00** | Despertar |
| **07:30–08:00** | Desayuno |
| **08:00–09:30** | Preparar mochila · agua · crema · baño · opcional: monestir / fotos frescas |
| **09:30–09:45** | Camino a estación **inferior** Funicular Sant Joan (pocos min desde monestir/hotel zona) |
| **~10:00** | **Primera subida** funicular → Sant Joan |
| **10:10** | Inicio ruta a pie → Sant Jeroni |
| **11:15–11:45** | Cima · fotos · snack |
| **11:45–13:00** | Bajada al funicular *(mismo camino)* |
| **~13:00–13:30** | Funicular abajo · comida / monestir |

**Si hace mucho calor:** cima antes de las 12:00 · agua extra · gorra.  
**Si queréis más largo:** bajar andando al monestir (Camí Vell) en vez de funicular vuelta → sumad ~1–1,5 h · billete **solo ida**.

## Cómo va la ruta a pie

1. Bajáis del funicular en **Sant Joan** (mirador / Pla de les Taràntules).  
2. Seguid señales **Sant Jeroni** (camino nuevo, bien marcado).  
3. Pasaréis zonas de encinas · cruce con Camí Vell · ermita/capilla zona Sant Jeroni.  
4. Últimos **escalones** a la cima · barandillas en tramos.  
5. Vuelta por el **mismo camino** al funicular *(opción simple)*  
   **o** Camí Vell directo al monestir *(más desnivel de bajada, sin 2º funicular)*.
