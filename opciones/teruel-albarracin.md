# Zona · Teruel · Sierra de Albarracín

Ramal **oeste/interior** (~2–2,5 h).  
Agua de río + patrimonio. Reglas calor → [README.md](README.md).

---

## Prioridad calor ★

| Prioridad | Plan |
|-----------|------|
| 1 | **Pozas** Hoz / Ebrón · chapuzón · **versión corta** |
| 2 | **Albarracín** casco (piedra / sombra calles) · piscina municipal si hay |
| 3 | Cascada Batida / Calicanto (paradas cortas) |
| ❌ | Rutas completas 5 h a mediodía |

**Dormir:** Albarracín hostal/casa · **A/C** · buen precio Booking.

## Qué ver / hacer

### Barranco de la Hoz *(Calomarde)* ★

| | |
|---|---|
| **Qué** | Cañón del río Blanco · pasarelas · pozas · arcos de piedra |
| **Inicio** | Calomarde (parking pueblo / salida) |
| **Tiempo** | Preferid **versión corta** ~**2–2,5 h** + baño · completa 3–4 h solo madrugada |
| **€** | **0** · sendero |
| **Agua** | Pozas para chapuzón · agua **fría** (~1300 m) |
| **Cerca** | Cascada Batida (muy corta · junto carretera) |

Calzado bueno · ropa baño · no con perros en pasarelas de rejilla (si aplica).

### Estrechos del río Ebrón *(El Cuervo – Tormón)* ★

| | |
|---|---|
| **Qué** | Pasarelas + puentes colgantes · cañón · **Cascada de Calicanto** |
| **Inicio** | El Cuervo (Teruel) · parking habilitado |
| **Tiempo** | Preferid **corta** · completa ~5 h solo si fresco |
| **€** | **0** |
| **Agua** | Remojar / chapuzón en puntos del cauce (caudal variable) |

*(No confundir con embalse Ebro de Fayón → [ebre-tarragona.md](ebre-tarragona.md))*

### Albarracín pueblo

- Casco medieval brutal · 1–2 h paseo  
- Buena base para dormir/cenar
