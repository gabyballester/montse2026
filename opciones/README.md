## Prioridad · calor extremo ★

1. **Agua** — baño, pozas, kayak/paddle corto, cascada, embalse, playa  
2. **Piscina** — alojamiento o municipal (siesta 14–17 h)  
3. **Fresco bajo techo** — iglesias, monasterios, museos, minas, cuevas cómodas, bodegas  
4. **Pueblos / patrimonio** — cascos, murallas, catas, edificios antiguos  
5. **Rutas a pie** — solo **ligeras y cortas** · madrugada · si >32 °C → saltad  

**Comer:** sombra · techado · A/C · baratos/bonitos con vistas si se puede.  
**Dormir:** **aire acondicionado** · buen precio · piscina si cabe.  
**Conductor:** cero alcohol al volante.

- Kayak / paddle / snorkel · **no muy largos ni duros** · con guía

- Comer: sombra, cubierto o local bonito barato con vistas
